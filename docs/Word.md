# 시세 데이터 관련 용어

| 용어 | 영어 | 설명 | 예시 / 비고 |
|---|---|---|---|
| 종목코드 | Symbol / Ticker / Code | 종목을 구분하기 위한 고유 코드 | 예: 삼성전자 종목코드 |
| 종목명 | Symbol Name | 종목의 이름 | 예: 삼성전자, SK하이닉스 |
| 현재가 | Current Price / Last Price | 가장 최근에 체결된 가격 | 화면에 가장 먼저 보이는 가격 |
| 시가 | Open | 해당 거래일에 처음 형성된 가격 | 장 시작 가격 |
| 고가 | High | 해당 기간 동안 가장 높았던 가격 | 일봉/분봉 데이터에 포함 |
| 저가 | Low | 해당 기간 동안 가장 낮았던 가격 | 일봉/분봉 데이터에 포함 |
| 종가 | Close | 해당 기간이 끝날 때의 가격 | 일봉에서는 당일 마감 가격 |
| 전일종가 | Previous Close | 전 거래일의 종가 | 등락률 계산 기준 |
| 대비 | Price Change | 전일종가 대비 현재가의 차이 | 상승/하락 금액 |
| 등락률 | Change Rate / Percent Change | 전일종가 대비 가격 변동 비율 | `%` 단위 |
| 거래량 | Volume | 일정 시간 동안 거래된 수량 | 주식 수 기준 |
| 거래대금 | Trading Value / Turnover | 거래된 총 금액 | 거래량 × 가격 |
| 체결가 | Execution Price / Trade Price | 실제로 거래가 성사된 가격 | 주문 가격과 다를 수 있음 |
| 체결량 | Execution Volume | 한 번의 체결에서 성사된 수량 | 틱 데이터에서 자주 사용 |
| 누적거래량 | Accumulated Volume | 장 시작 후 누적된 전체 거래량 | 장중 변화 확인에 사용 |
| 누적거래대금 | Accumulated Trading Value | 장 시작 후 누적된 거래 금액 | 거래 활발한 종목 판단에 도움 |
| 호가 | Bid/Ask Quote | 매수/매도 대기 가격 | 호가창에서 확인 |
| 매수호가 | Bid Price | 사려는 쪽이 제시한 가격 | 가장 높은 매수호가가 중요 |
| 매도호가 | Ask Price | 팔려는 쪽이 제시한 가격 | 가장 낮은 매도호가가 중요 |
| 호가잔량 | Bid/Ask Size | 각 호가 구간에 쌓여 있는 수량 | 매수/매도 대기 물량 |
| 매수잔량 | Bid Size | 매수호가에 쌓인 수량 | 수급 판단 참고 |
| 매도잔량 | Ask Size | 매도호가에 쌓인 수량 | 저항 물량 판단 참고 |
| 스프레드 | Spread | 최우선 매도호가와 최우선 매수호가의 차이 | 좁을수록 거래가 유리한 편 |
| 틱 | Tick | 가격 또는 체결 정보의 최소 변화 단위 | 틱 차트, 틱 데이터 |
| 틱 데이터 | Tick Data | 체결이 발생할 때마다 쌓이는 세밀한 데이터 | 초단타 전략에서 중요 |
| 봉 | Candle / Bar | 일정 시간 동안의 가격 흐름을 묶은 데이터 | 시가/고가/저가/종가 포함 |
| 분봉 | Minute Candle | 분 단위로 묶은 봉 데이터 | 예: 1분봉, 5분봉 |
| 일봉 | Daily Candle | 하루 단위 봉 데이터 | 가장 기본적인 차트 단위 |
| 주봉 | Weekly Candle | 일주일 단위 봉 데이터 | 중기 추세 확인 |
| 월봉 | Monthly Candle | 한 달 단위 봉 데이터 | 장기 추세 확인 |
| OHLC | Open, High, Low, Close | 시가, 고가, 저가, 종가 4개 가격 데이터 묶음 | 차트 데이터 기본 구조 |
| 체결시간 | Execution Time / Trade Time | 거래가 실제로 성사된 시각 | 초단위까지 제공되기도 함 |
| 시세시간 | Quote Time / Timestamp | 시세 데이터가 기록된 시각 | 실시간/과거 데이터 구분에 중요 |
| 기준가 | Reference Price | 비교 기준이 되는 가격 | 전일종가나 기준 시가 등 |
| 상한가 | Upper Limit Price | 하루 동안 허용되는 최대 상승 가격 | 시장 규칙에 따라 결정 |
| 하한가 | Lower Limit Price | 하루 동안 허용되는 최대 하락 가격 | 시장 규칙에 따라 결정 |
| VI | Volatility Interruption | 급격한 가격 변동 시 단일가 매매로 전환되는 제도 | 변동성 완화 장치 |
| 예상체결가 | Indicative Match Price | 동시호가 등에서 예상되는 체결 가격 | 실제 체결가와 다를 수 있음 |
| 동시호가 | Call Auction | 일정 시간 동안 주문을 모아 한 번에 가격을 결정하는 방식 | 장 시작 전/마감 전 자주 등장 |
| 수정주가 | Adjusted Price | 액면분할, 배당 등 권리 변동을 반영해 보정한 가격 | 과거 차트 분석 시 중요 |
| 원주가 | Raw Price / Unadjusted Price | 보정이 적용되지 않은 원래 가격 | 실제 당시 가격 그대로 |
